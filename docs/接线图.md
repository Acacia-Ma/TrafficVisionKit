# ATK-MC5640 与 STM32F407 接线图

ATK-MC5640 模块可直接与正点原子探索者 STM32F407 开发板的 CAMERA 接口连接，并通过 SCCB 协议通信。  
根据现有接线图，整理后的引脚对应关系如下：

## 引脚一对一对应（横线格式）

3.3V - V3.3  
VSYNC - PB7  
HREF - PA4  
RST - PG15  
D1 - PC7  
D3 - PC9  
D5 - PB6  
D7 - PE6  
FLASH - PA8  

GND - GND  
SCL - PD6  
SDA - PD7  
D0 - PC6  
D2 - PC8  
D4 - PC11  
D6 - PE5  
PCLK - PA6  
PWDN - PG9
